# Task-04-Zainab-Younas# Sales Data Analysis Project (Power BI)

An end-to-end data analysis project focused on cleaning, transforming, and visualizing sales data using Power BI to uncover yearly revenue trends and operational insights.

## 🚀 Features & Workflow
* **Data Cleaning & ETL:** Handled missing values, formatted dates, and structured raw order data using Power Query.
* **Data Modeling:** Organized tables to analyze performance across products, order statuses, and referral sources.
* **Key Metrics Tracked:** Total Sales, Order Volume, and Customer Breakdown.

## 🛠️ Tools Used
* **Power BI / Power Query** - For data transformation and dashboard creation.
* **Excel / CSV** - Data source containing order details, payment methods, and tracking info.
